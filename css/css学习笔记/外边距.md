## 外边距（margin）

CSS Margin(外边距)属性定义元素周围的空间

CSS Margin属性接受任何长度单位、百分数值甚至负值

margin可以单独改变元素的上，下，左，右边距。也可以一次改变所有的属性

| 值       | 说明                                       |
| -------- | ------------------------------------------ |
| auto     | 设置浏览器边距。这样做的结果会依赖于浏览器 |
| *length* | 定义一个固定的margin（使用像素，pt，em等） |
| *%*      | 定义一个使用百分比的边距                   |
