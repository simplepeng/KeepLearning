# 封装

`封装`也叫`数据隐藏`是面向对象编程中一个重要的概念。

类包含`私有成员（private`）、`保护成员（protected)`和`公有成员（public）`成员。

```c++
class Box
{
   public://公有
      double getVolume(void)
      {
         return length * breadth * height;
      }
   private://私有
      double length;      // 长度
      double breadth;     // 宽度
      double height;      // 高度
};
```
