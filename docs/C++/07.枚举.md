# 枚举

枚举类型(enumeration)是C++中的一种派生数据类型，它是由用户定义的若干枚举常量的集合。

创建枚举，需要使用关键字 `enum`

```c++
enum 枚举名{ 
     标识符[=整型常数], 
     标识符[=整型常数], 
... 
    标识符[=整型常数]
} 枚举变量;
```

```c++
enum color { red, green, blue } c;
c = blue;
//red = 0
//greem = 1
```

默认情况下，第一个名称的值为 0，后续值依次`+1`

但是，也可以给名称赋予一个特殊的值，只需要添加一个初始值即可

```c++
enum color { red, green=5, blue };
//green = 5
//blue = 6
```
