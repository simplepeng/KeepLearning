# 列表

列表(list)可以将多个元素放在一起，使用方括号`[]`标注

```python
list1 = [1,2,3]
```

列表可以包含不同类型的元素，但一般情况下，元素类型都相同

```python
list2 = [1,2,3,"4"]
```

`append()`函数能在列表末尾添加新的元素

```python
list1.append(5)
```

`len()`函数也支持列表

```python
len = len(list1)
print(len)
```

列表的常用函数

```python
list.remove(x) 
list.clear()
list.index()
list.count(x)
list.sort()
list.reverse()
list.copy()
```
