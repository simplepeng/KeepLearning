# 变量

变量是一个数据存储位置，其值在程序执行期间可以改变

```c
//声明变量
变量类型 变量名;
//变量赋值
变量名 = 值;
```

* 变量的名称可以由字母、数字和下划线字符组成
* 数字不能作为变量名开始
* C是大小写敏感的，所以变量`Person`和`person`是两个变量
* 尽量遵循驼峰命名法
* 关键字不能作为变量名

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a;
    a = 1;
    printf("a == %d\n", a);
    char b = 'c';
    printf("a == %d\n", b);
    return 0;
}
```

## 变量作用域

* 局部变量：在函数内定义的参数
* 全局变量：在所有函数外定义的参数
